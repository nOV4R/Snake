              ПРОГРАММА "ЗМЕЙКА"

НАЗНАЧЕНИЕ:
-----------
Программа реализует движение змейки по экрану
с отражением от границ по законам отражения света.

ОСНОВНЫЕ ФУНКЦИИ:
----------------
• Случайное начальное направление движения
• Отражение от границ экрана
• Изменение скорости пользователем
• Информационная панель с параметрами
• Режим паузы

УПРАВЛЕНИЕ:
-----------
1        - Увеличить скорость
2        - Уменьшить скорость
ПРОБЕЛ   - Пауза/Продолжить
ESC      - Выход


СОСТАВ ПРОЕКТА:
---------------
Snake.java        - Основной класс игры
SnakeSegment.java - Класс сегмента змейки  
Constants.java    - Класс констант
manifest.mf      - Манифест для JAR-файла
README.md        - Эта инструкция

СБОРКА И ЗАПУСК:
---------------
1. Через консоль перейти в папку проекта

2. Компиляция:
   javac -d out -encoding UTF-8 src/*.java

3. Создание JAR:
   jar cfm Snake.jar manifest.mf -C out . 
   (обратите внимание на точку в конце команды)

4. Запуск:
   java -jar Snake.jar"
